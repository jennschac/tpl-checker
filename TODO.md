# TPL Checker — Feature Ideas

## Expand page support
- Support Goodreads **bookshelf pages** (`goodreads.com/review/list/*`) in addition to blog pages
- Support other Goodreads list formats (listopia, user lists, etc.)

## Borrow directly from the badge
- Add a "Borrow" or "Place Hold" action to the badge that takes you directly to the TPL checkout/hold flow, rather than just linking to the catalogue record

## Format toggle
- Let users choose which formats to check availability for:
  - eBook (currently the only one)
  - eAudiobook
  - Physical book
- Could be a popup settings UI or a toggle that persists across sessions
- Badge would reflect the selected format(s)
